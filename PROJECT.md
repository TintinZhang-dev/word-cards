# WordCards — 项目规划

> 目标：类似 Anki 的智能单词卡工具
> 创始：2026-06-25 | 当前状态：Phase 2（SM-2 + 学习模式 + 卡片状态管理）

## 🎯 最终愿景

一个可以通过上传文档自动提取生词、按记忆曲线安排复习、帮用户掌握语言的智能卡片工具。最终可发展为独立应用。

- [x] 3D 翻转卡片（纯 HTML/CSS/JS）

## 参考资料
- **Anki** — 核心参考，间隔重复（Spaced Repetition）
- **SM-2 算法** — SuperMemo 算法，Anki 基础
- **Leitner System** — 另一种卡片盒复习法
